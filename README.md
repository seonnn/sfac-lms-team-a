# LMS 수강시스템

## 동영상 시청, 과제 제출, 커뮤니티 기능을 갖춘 LMS 시스템

> 프로젝트 기간: 2023년 7월 17일 ~ 2023년 8월 11일 (26일)

[배포링크 바로가기](https://withpet-two.vercel.app/)

## 팀 소개

### 저희 팀은 이런 기능을 담당했습니다!

> LMS 5가지 분류 중 분류 3번의 과제제출/평가시스템 제작을 담당하였습니다.
>
> 멘토와 수강생의 과제공유 솔루션으로, 멘토의 과제 생성 기능과 수강생의 과제 제출 기능 구현을 통한 양방향 과제제출/평가시스템을 개발하였습니다.
